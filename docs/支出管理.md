# 機能仕様：支出管理（CRUD）

## 概要
DBに保存された支出データの登録・取得・編集・削除を行う。

## 主要機能
- **保存**: 確認済みデータをDBに登録する。
- **一覧**: Reactの `useEffect` でデータを取得し、カード形式またはテーブル形式で表示する。
- **詳細**: 特定の支出データを1件表示する。
- **編集**: 特定の支出ID（PK）を指定して内容を更新する。
- **削除**: 特定の支出ID（PK）を指定して削除する。

## 保存する主な項目
- 店名
- 購入日
- 合計金額
- カテゴリー
- レシート画像
- OCRで読み取った全文
- 作成日時
- 更新日時

## エンドポイント案
- `GET /api/expenses/`: 支出一覧取得
- `POST /api/expenses/`: 支出データ保存
- `GET /api/expenses/<int:id>/`: 支出詳細取得
- `PUT /api/expenses/<int:id>/`: 支出編集
- `DELETE /api/expenses/<int:id>/`: 支出削除
